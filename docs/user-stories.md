# Smart Clinic Management System – User Stories

## Doctor
- As a Doctor, I want to log in securely so that I can access my schedule.
- As a Doctor, I want to set my available time slots so that patients can book appointments.
- As a Doctor, I want to view my appointments for a selected date so that I can plan my day.
- As a Doctor, I want to view basic patient details for an appointment so that I can provide care.
- As a Doctor, I want to create a prescription for a patient so that treatment is recorded.

## Patient
- As a Patient, I want to register and log in securely so that I can book appointments.
- As a Patient, I want to search for doctors by name or specialty so that I can find the right doctor.
- As a Patient, I want to view a doctor’s available time slots for a date so that I can choose a time.
- As a Patient, I want to book an appointment so that I can visit the clinic.
- As a Patient, I want to view my appointments so that I can track upcoming/past visits.
- As a Patient, I want to view my prescriptions so that I can follow treatment.

## Admin
- As an Admin, I want to log in to the admin portal so that I can manage the system.
- As an Admin, I want to add doctors so that they can serve patients.
- As an Admin, I want to update doctor information so that the directory is accurate.
- As an Admin, I want to manage patient accounts so that the system stays clean.
- As an Admin, I want to view reporting data so that the clinic can track activity.
